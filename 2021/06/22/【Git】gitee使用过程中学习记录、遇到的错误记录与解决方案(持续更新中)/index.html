学习记录

Git基本操作如下：

1.远程建立仓库（在Git网站里）
仓库名称、仓库介绍、选择语言、模板不选择、使用read.me文件初始化这个仓库，分支模型选择单分支

2.本地建立仓库
自己建立一个文件夹后，打开文件夹
右键选择 git bash
方法一（常用）：
git init 初始化
git remote add origin +链接 取得联系
git pull origin master 把远程仓库的代码拉下来
方法二：
git clone
生成本地库，把远程库的包括文件夹拉下来

3.本地库提交到远程库
git status 获取当前状态
git add xxx文件
git add . 提交当前目录所有文件
git commit -m “Add:xxx备注”
git push origin master
